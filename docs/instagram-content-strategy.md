## 정기 시리즈

### 1. 오늘의 트렌딩 TOP 3
- **빈도**: 매일 or 주 3회
- **포맷**: 카드뉴스 (3장 슬라이드)
- **소재**: 트렌딩 홈 복합 점수 상위 3개 제품
- **앵글**: "복합 점수 기준 오늘 가장 핫한 뷰티 제품"

**카드 구성 예시**
```
[1장] 날짜 + "오늘의 트렌딩"
[2장] #1 제품 — 브랜드명 / 제품명 / 티어 배지 / 점수
      #2 제품 — 브랜드명 / 제품명 / 티어 배지 / 점수
      #3 제품 — 브랜드명 / 제품명 / 티어 배지 / 점수
[3장] "매일 업데이트 되는 전체 순위는 👉 링크 인 바이오"
```

**캡션 예시**
```
오늘의 뷰티 트렌딩 🌿

1위 [브랜드] [제품명] — 확정 트렌드
2위 [브랜드] [제품명] — 버즈 집중
3위 [브랜드] [제품명] — 신규 감지

올영 랭킹 + 유튜브 버즈 + 네이버 검색량을
종합한 복합 점수 기준이에요.

전체 TOP 20 → 링크 인 바이오

#뷰티트렌드 #클리키 #오늘의화장품 #뷰티크리에이터
```

### 2. 신규 감지 알림
- **빈도**: 주 2회 (화 / 금 권장)
- **포맷**: 단일 이미지 or 카드 2장
- **소재**: `newly_discovered` 티어 제품 (검색량 급등, 유튜브 미진입)
- **앵글**: "아직 아무도 안 만든 콘텐츠 소재"

**카드 구성 예시**
```
[1장] "📡 신규 감지됨" + 제품명 + 브랜드
[2장] "이 제품 유튜브 영상: 0개 / 네이버 검색 증가율: +XX%
       먼저 만들면 조회수 독식 가능"
```

**캡션 예시**
```
📡 신규 감지 알림

[제품명]

네이버 검색량이 7일 만에 급등했는데
아직 유튜브 영상이 거의 없어요.

지금 만들면 선점 가능한 소재예요 👀

#신규감지 #콘텐츠소재 #뷰티크리에이터 #클리키
```

---

### 3. 버즈 vs 실구매 괴리 분석
- **빈도**: 주 1회 (수요일 권장)
- **포맷**: 카드뉴스 3~4장
- **소재**: `buzz_only` 티어 제품 (유튜브 언급 많음 + 올영 랭킹 반영 안 됨)
- **앵글**: "SNS에서 뜨는데 실제로 잘 팔릴까?"

**카드 구성 예시**
```
[1장] "SNS vs 실구매 — 이 제품 어디까지 왔나"
[2장] 제품명 + 유튜브 버즈 점수 (높음) vs 올영 랭킹 변동 (없음)
[3장] "아직 구매 전환 안 됨 = 기회인가, 거품인가"
[4장] 클리키에서 오가닉 비율 확인하는 법
```

**캡션 예시**
```
유튜브에서 엄청 뜨는데 올영 랭킹엔 없는 제품들 🤔

[제품명]
→ 유튜브 버즈 점수: 높음
→ 올영 7일 랭킹 변동: 없음
→ 오가닉 비율: XX%

광고성 버즈인지, 진짜 트렌드인지
클리키에서 오가닉 비율로 구분할 수 있어요.

#뷰티분석 #트렌드분석 #클리키 #뷰티마케팅
```

### 4. 주간 브랜드 점유율
- **빈도**: 주 1회 (월요일 권장)
- **포맷**: 인포그래픽 단일 이미지
- **소재**: TOP 20 제품에서 브랜드별 등장 횟수
- **앵글**: "이번 주 트렌딩을 지배한 브랜드는?"

**카드 구성 예시**
```
[단일] 가로 바 차트 형태
       브랜드명 | ████████ 5개
       브랜드명 | ██████   3개
       브랜드명 | ████     2개
       ...
```

**캡션 예시**
```
이번 주 트렌딩 TOP 20, 브랜드별로 보면 📊

1위 [브랜드] — X개 제품 진입
2위 [브랜드] — X개 제품 진입
3위 [브랜드] — X개 제품 진입

지난주 대비 새로 진입한 브랜드: [브랜드명]

#주간트렌드 #뷰티브랜드 #클리키 #뷰티인사이트
```

## 기획성 콘텐츠 (월 2회)

### 5. "광고 걷어낸 진짜 트렌드"
- **빈도**: 월 2회
- **포맷**: 카드뉴스 4장
- **소재**: 오가닉 buzz ratio 높은 제품만 필터링
- **앵글**: "광고 없이 진짜 뜨는 제품은 다르다"

**카드 구성 예시**
```
[1장] "광고 제외하고 보면 순위가 바뀐다"
[2장] 광고 포함 TOP 5 vs 오가닉만 TOP 5 비교표
[3장] 오가닉 비율 높은 제품 소개 (이 제품은 진짜 입소문)
[4장] "클리키에서 광고 포스트 제외 토글 → 순수 버즈만"
```

---

### 6. 트렌드 차트 스토리 (릴스)
- **빈도**: 월 2회
- **포맷**: 릴스 15~30초
- **소재**: 특정 제품의 7일 점수 추이 차트
- **앵글**: "이 제품 왜 갑자기 터졌나"

**릴스 구성 예시**
```
0~3초  : 제품명 + "왜 갑자기 터졌을까?"
4~12초 : 7일 차트 라인이 올라가는 애니메이션
13~20초: 원인 분석 (특정 크리에이터 영상 업로드 → 검색량 폭발)
21~30초: "이런 인사이트 매일 → 클리키 링크 인 바이오"
```
